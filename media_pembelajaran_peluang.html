▶️ Tonton Video Tebak Peluang (YouTube Shorts)

🎲 Ruang Sampel (S)

Himpunan semua hasil yang mungkin terjadi dari suatu percobaan acak.

Koin: S = {A, G} AG
Dadu: S = {1,2,3,4,5,6} 123456

📌 Titik Sampel

Setiap anggota dari ruang sampel. Satu kemungkinan hasil tunggal dari percobaan.

Contoh: Pada dadu, titik sampel "3" berarti dadu berhenti tepat di angka 3.

n(S) = banyaknya titik sampel

📐 Peluang Teoretik

Peluang yang dihitung berdasarkan logika/rumus matematika, bukan percobaan nyata.

P(A) = n(A) / n(S)
n(A) = banyak kejadian A • n(S) = banyak ruang sampel

📊 Peluang Empirik

Peluang yang diperoleh dari data hasil percobaan nyata (frekuensi relatif).

P(A) = f(A) / n
f(A) = frekuensi kejadian A • n = banyak percobaan

💡 Ingat: Nilai peluang selalu berada antara 0 ≤ P(A) ≤ 1. Jika P(A) = 0 → mustahil. Jika P(A) = 1 → pasti terjadi. Di antaranya → mungkin terjadi.

✅ Evaluasi Formatif
Uji Pemahamanmu!

3 soal pilihan ganda interaktif — pilih jawaban terbaikmu

Soal 1 dari 3
Sebuah dadu dilempar satu kali. Berapa peluang teoritis munculnya angka genap?
✅ Pembahasan: Ruang sampel dadu S = {1,2,3,4,5,6}, n(S)=6. Kejadian angka genap A = {2,4,6}, n(A)=3. Maka P(A) = 3/6 = 1/2. Jawaban C benar!
Soal 2 dari 3
Dalam sebuah kotak terdapat 4 bola merah dan 6 bola biru. Jika diambil satu bola secara acak, peluang terambilnya bola merah adalah...
✅ Pembahasan: Total bola n(S) = 4+6 = 10. Bola merah n(A) = 4. Maka P(merah) = 4/10 = 2/5. Jawaban A benar!
Soal 3 dari 3
Sebuah koin dilempar sebanyak 50 kali, dan sisi Angka muncul 18 kali. Berapakah peluang empirik munculnya sisi Angka?
✅ Pembahasan: Peluang empirik = frekuensi kejadian / total percobaan = 18/50 = 9/25 = 0,36. Ini berbeda dari teori (0,5) karena percobaan nyata memiliki variasi alami. Jawaban B benar!

📚 Glosarium Istilah Penting

Peluang
Ukuran matematis tentang seberapa besar kemungkinan suatu kejadian terjadi, nilainya antara 0 (mustahil) sampai 1 (pasti).
Ruang Sampel (S)
Himpunan semua hasil yang mungkin terjadi dari suatu percobaan acak. Dilambangkan dengan S.
Titik Sampel
Setiap anggota dari ruang sampel; satu kemungkinan hasil tunggal dari sebuah percobaan.
Peluang Teoretik
Peluang yang dihitung menggunakan logika matematika: P(A) = n(A)/n(S), tanpa melakukan percobaan fisik.
Peluang Empirik
Peluang yang ditentukan berdasarkan data hasil percobaan nyata (frekuensi relatif): P(A) = f(A)/n.
Frekuensi Relatif
Perbandingan antara banyaknya suatu kejadian terjadi dengan total seluruh percobaan yang dilakukan.
